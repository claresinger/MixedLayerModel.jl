Entrainment

MixedLayerModel.sv_jump — Method
sv_jump(u, p, LWP)

Δsv = Δs + CpΔTv - CpΔT
    = Δs + Cp(Rv/Rd - 1)(Tft*qft - T_zi*qM) + Cp(Rv/Rd)(T_zi*ql_zi)

Tv = (Rd*(1-qt) + Rv*qv)/Rd * T

jump in virtual liquid static energy across inversion
proportional to buoyancy jump
used in energy balance entrainment
source
MixedLayerModel.we — Method
we(u, p, zb, LWP, etype::bflux)

entrainment velocity based on buoyancy flux
without radiation

integral is calculated analytically
source
MixedLayerModel.we — Method
we(u, p, zb, LWP, etype::enBal)

entrainment velocity obtained via energy balance requirement
w = ΔR / (Δsv * ρref)
Δsv = the jump in virtual liquid static energy
sv = Cp*Tv + g*z - Lv*ql = s + Cp(Tv - T)
source

Cloud fraction

MixedLayerModel.calc_decoupling — Method
calc_decoupling(u, p)

calculates the stability parameter, s
also called the minimal decloupling parameter
in Bretherton and Wyant (1997)
De = (LHF / ΔR) * (zc / zi)
where zc is the cloud thickness (zi-zb)
source
MixedLayerModel.cloud_fraction — Method
cloud_fraction(u, p, zb, LWP)

calculates the cloud fraction 
inspired by Chung et al 2012
uses a logistic function to smoothly interpolate between 
100% and 20% cloud fraction based on the value of the
stability parameter
source

MLM ODE

MixedLayerModel.mlm — Method
mlm(du, u, p, t)    

define the coupled ODE
  dzi/dt = we - D*zi
  dsM/dt = -dE/dz = -1/zi * (Szi - S0 + dR/rho)
  dqM/dt = -dW/dz = -1/zi * (Qzi - Q0)
  dSST/dt = (SWnet - LWnet - SHF - LHF - OHU) / τ_SST
  dCF/dt = (CF' - CF) / τ_CF
source
