Entrainment

MixedLayerModel.sv_jump — Method
sv_jump(u, p, LWP)

calculate the jump in virtual liquid static energy (sv)
across the inversion
Δsv = Δs + CpΔTv - CpΔT
Tv = (Rd*(1-qt) + Rv*qv)/Rd * T

MLM ODE

MixedLayerModel.mlm — Method
mlm(du, u, p, t)    

define the coupled ODE
  dzi/dt = we - D*zi
  dhM/dt = -dE/dz = 1/zi * (Hzi - H0 + dR/rho)
  dqM/dt = -dW/dz = 1/zi * (Qzi - Q0)
  dSST/dt = 1/c * (SWnet - LWnet - SHF - LHF - OHU)
source
